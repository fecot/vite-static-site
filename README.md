# Viteでの静的サイト構築環境

## 概要

[Vite](https://vitejs.dev/)で静的サイト環境を構築するためのテンプレートです。

## 使用パッケージ

1. tailwindcss
2. sass (autoprefixer, postcss)
3. html-handlebars (TODO)
4. prettier (TODO)
5. eslint (TODO)

## 開発方法

ローカル環境立ち上げ

```terminal
npm install
npm run dev
```

ビルド (distディレクトリに出力

```terminal
npm run build
```
